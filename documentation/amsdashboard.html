AMS Dashboard


1. Feature Overview


AMS Dashboard is a php web application which shows all data that has been logged by the AMS Android Library. The data used in the AMS Dashboard is pulled from the backend database.

The data will be displayed using tables and graphs depending on which is more user friendly.


Goals:

  • Display data in a relevant, user friendly way

  • Include the ability to create favorite graphs

  • Dashboard will have a left navigation menu

  • Dashboard should be able to connect via local network if the user specifies
